Codify é agora Open Source

Código disponível no GitHub sob licença MIT. Contribuições são bem-vindas!

É com grande entusiasmo que anunciamos que o Codify é agora Open Source! O código fonte está disponível no GitHub sob a licença MIT, permitindo que qualquer pessoa use, modifique e contribua para o projeto.

Porquê Open Source?

A decisão de abrir o código do Codify foi tomada por várias razões:

  • Transparência — os utilizadores podem ver exatamente como o IDE funciona, como os dados são processados e como a IA opera localmente
  • Comunidade — acreditamos que uma comunidade ativa torna o produto melhor para todos
  • Inovação — desenvolvedores de todo o mundo podem contribuir com ideias, correções e funcionalidades
  • Segurança — código aberto permite auditoria independente e identificação rápida de vulnerabilidades
  • Educação — o Codify pode ser usado como recurso de aprendizagem para quem quer perceber como um IDE moderno é construído

O que está incluído

O repositório no GitHub inclui:

  • Core do editor — motor de rendering, gestão de buffers, syntax highlighting
  • Neuria AI — motor de IA local com suporte a múltiplos modelos
  • Interface do utilizador — toda a UI construída com componentes modernos
  • Sistema de extensões — API para criar e publicar extensões
  • Terminal integrado — emulador de terminal completo
  • Suporte Git — interface visual para controlo de versões
  • Build scripts — scripts para compilar o Codify em Windows, macOS e Linux

Licença MIT

Escolhemos a licença MIT por ser a mais permissiva e amigável para a comunidade. Podes:

  • Usar o Codify para qualquer fim, incluindo comercial
  • Modificar o código e distribuir as tuas próprias versões
  • Incorporar partes do Codify nos teus projetos
  • Criar forks e explorar direções diferentes

A única exigência é manter o aviso de copyright e a licença nas tuas distribuições.

Como Contribuir

Se queres contribuir para o Codify, aqui estão algumas formas de o fazer:

  • Reporta bugs — abre uma issue no GitHub com uma descrição clara do problema
  • Sugere funcionalidades — usa o GitHub Discussions para propor novas ideias
  • Submete PRs — corrige bugs, melhora a documentação ou adiciona funcionalidades
  • Traduções — ajuda a traduzir o Codify para mais idiomas
  • Extensões — cria e publica extensões no marketplace

O Futuro

Esta é apenas a primeira etapa. Com a comunidade envolvida, planeamos acelerar o desenvolvimento e trazer funcionalidades ainda mais ambiciosas. O Codify continuará a ser gratuito e open source, com uma versão enterprise opcional para equipas que necessitem de suporte dedicado.

Visita o nosso repositório no GitHub para explorar o código, contribuir ou simplesmente deixar uma estrela. Toda a ajuda é bem-vinda!
